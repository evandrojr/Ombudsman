COMO CONSULTAR UMA RESPOSTA :

- Basta preencher qualquer valor nos campos PROTOCOLO e CODIGO DE ACESSO

COMO FAZER LOGIN NA INTRANET :

- Basta preencher um dos seguintes valores no campo LOGIN e qualquer valor no campo SENHA

  1. Para logar como ADMINISTADOR, digite administrador
  2. Para logar como ATENDENTE, digite atendente
  3. Para logar como ESPECIALISTA, digite especialista
  4. Para logar como GESTOR DE CONTEUDO, digite conteudo
  5. Para logar como GESTOR DE SISTEMAS, digite sistemas
  6. Para logar como OUVIDOR, digite ouvidor
  7. Para logar como OUVIDOR GERAL, digite ouvidorgeral


- Esse "Ajuda" traz apenas informações de como acessar partes específicas desse protótipo. Serve apenas para dar um exemplo do layout e auxiliar na navegação do protótipo. O objetivo do "Ajuda" no sistema será auxiliar na navegação do mesmo, explicando as funcionalidades e como utilizá-las.
